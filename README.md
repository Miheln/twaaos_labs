# TWAAOS - laboratoare Iuliana

Acest proiect contine laboratoarele TWAAOS lucrate separat, cate un folder pentru fiecare laborator.

Materialele primite sunt in folderul `docs`.

## Pornire

Intra in folderul laboratorului dorit si ruleaza:

```bash
./start.sh
```

Fiecare laborator este pastrat separat ca sa poata fi verificat independent.
